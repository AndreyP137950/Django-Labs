# Подсказка по использованным библиотекам (для защиты лабораторной)

В рамках данной лабораторной работы (№5) мы использовали три основные внешние библиотеки. Ниже приведено краткое объяснение каждой из них, которое поможет вам ответить на вопросы преподавателя при защите.

---

### 1. **`pyTelegramBotAPI` (часто называют `telebot`)**
- **Для чего нужна:** Это современный и удобный фреймворк для взаимодействия с Telegram Bot API. Она реализует всю логику общения программы с серверами Telegram: прием сообщений, отправка ответов, создание кнопок.
- **Где используется:** 
  В файле `main.py`. С её помощью:
  - Инициализируется бот (`bot = telebot.TeleBot(BOT_TOKEN)`).
  - Создаются интерфейсные текстовые клавиатуры (`ReplyKeyboardMarkup`, `KeyboardButton`).
  - Определяются "обработчики" (декораторы `@bot.message_handler`), которые реагируют, когда пользователь пишет команду (например, `/start`) или нажимает на кнопку с текстом (например, "Курс USD").

### 2. **`requests`**
- **Для чего нужна:** Это одна из самых популярных библиотек Python для отправки HTTP-запросов (в нашем случае GET-запроса) к целевым сайтам в интернете поверх протокола TCP/IP.
- **Где используется:** 
  В файле `cbr_parser.py`.
  С её помощью скрипт обращается на прямую гиперссылку банка за свежими данными (`requests.get(CBR_DAILY_URL, ...)`). Она "притворяется" реальным браузером (мы передаем ей специальные заголовки `User-Agent`) и скачивает сырой HTML-код со страницы Центробанка, после чего передает этот скаченный текст для дальнейшей обработки.

### 3. **`beautifulsoup4` (часто импортируется как `bs4`)**
- **Для чего нужна:** Библиотека для синтаксического разбора (парсинга) разметки. Скачанный библиотекой `requests` текст — это длинная, сложная строка HTML, из которой очень трудно извлечь нужную информацию обычными средствами (например, регулярными выражениями). `BeautifulSoup` превращает этот хаотичный текст в структурированное дерево (Объектную Модель Документа — DOM).
- **Где используется:** 
  В файле `cbr_parser.py`.
  С её помощью мы:
  1. Читаем структуру страницы (`BeautifulSoup(response.text, 'html.parser')`).
  2. Легко находим таблицу с курсами валют по её HTML-тегу и классу (`soup.find('table', class_='data')`).
  3. Проходим циклом (`for row in rows...`) по каждой строке таблицы и вытягиваем конкретные значения (код валюты, её имя, актуальный курс ЦБ).
