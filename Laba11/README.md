# Лабораторная работа 11: Telegram CASE-бот

## Назначение
Программа реализует мини CASE-систему в формате Telegram-бота.
Бот позволяет моделировать выражение:

y = F1(F2(F3(x)))

Доступные функции:
- sqrt
- reciprocal (1/x)
- exp (e^x)

Бот умеет:
- выбирать функции F1, F2, F3;
- вычислять результат для заданного x;
- проверять область определения функций и выводить ошибки;
- генерировать исходный код на VBA для Excel;
- выполнять выбор операций через кнопки в интерфейсе Telegram.

## Структура основных файлов
- run_bot.py: точка входа для запуска бота.
- .env: хранение BOT_TOKEN.
- requirements.txt: зависимости проекта.
- telegram_case_bot/config.py: загрузка токена из окружения.
- telegram_case_bot/bot.py: команды и обработчики Telegram-бота.
- telegram_case_bot/functions.py: математическая логика и проверки.
- telegram_case_bot/vba_generator.py: генерация VBA кода.
- telegram_case_bot/storage.py: хранение конфигурации пользователя (в памяти).

## Установка и запуск
1. Установите зависимости:
   py -3.11 -m pip install -r requirements.txt

2. Укажите токен бота в файле .env:
   BOT_TOKEN=ваш_токен_от_BotFather

3. Запустите бота:
   py -3.11 run_bot.py

Если токен не указан, программа завершится с понятной ошибкой.

## Команды бота
- /start: запуск и краткая инструкция.
- /help: список команд.
- /show: показать текущие F1, F2, F3 и формулу.
- /setf1 <sqrt|reciprocal|exp>: установить F1.
- /setf2 <sqrt|reciprocal|exp>: установить F2.
- /setf3 <sqrt|reciprocal|exp>: установить F3.
- /calc <x>: вычислить y = F1(F2(F3(x))).
- /code: получить сгенерированный VBA-код.

## Кнопочный режим
После команды /start в чате появляется клавиатура с кнопками:
- Показать конфигурацию
- Выбрать F1
- Выбрать F2
- Выбрать F3
- Вычислить
- Сгенерировать VBA
- Помощь

Кнопки Выбрать F1/F2/F3 открывают inline-кнопки для выбора функции:
sqrt, reciprocal, exp.

Кнопка Вычислить переводит бота в режим ожидания x: 
после нажатия нужно отправить число отдельным сообщением.

## Пример работы
1. /setf3 exp
2. /setf2 reciprocal
3. /setf1 sqrt
4. /calc 2

Бот последовательно посчитает:
- F3(2) = e^2
- F2(...) = 1 / e^2
- F1(...) = sqrt(1 / e^2)

И отправит результат вместе с шагами вычисления.

## Обработка ошибок области определения
- Для sqrt(x): x должен быть >= 0.
- Для reciprocal(x): x не должен быть равен 0.
- Для exp(x): ограничений по области определения нет.

При нарушении условий бот возвращает текст ошибки.

## Генерация VBA
Команда /code генерирует VBA-код с функциями:
- sqrt_func
- reciprocal_func
- exp_func
- CalculateChain
- TestCalculate

Этот код можно вставить в модуль VBA редактора Excel.

## Важно
Текущее хранилище конфигурации пользователя работает в памяти процесса.
После перезапуска бота выбранные F1/F2/F3 сбрасываются к значениям по умолчанию.
